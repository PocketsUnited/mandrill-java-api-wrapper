Welcome to MandrillJavaAPIWrapper

This is currently in early beta status as only a small part of the whole API is implemented

Goal

MandrillJavaAPIWrapper provides an easier access to the Mandrill email sending service when building a Java project. It provides a builder style syntax to create requests for the various endpoints. It offers service class implementations that can have some default values e.g. for your Mandrill API key. It is ready to be used within a Spring application

Status

Currently only a subset of the API endpoints is covered. A full coverage is planned for the 1.0.0 release
